# Project Summary
The F1 Fan Analytics Platform is an advanced application tailored for Formula 1 enthusiasts, offering real-time insights, analytics, and predictions about races, drivers, and constructors. With a user-friendly interface and personalized profiles, the platform leverages AI and machine learning to enhance user engagement and provide valuable data for both casual fans and serious analysts. It integrates real-time data from the Ergast API, making it an essential tool for anyone passionate about Formula 1 racing.

# Project Module Description
- **User Account Management**: Handles user login, signup, profile management, and settings.
- **Dashboard**: Displays real-time race statistics, standings, and analytics.
- **AI Assistant**: Provides intelligent support and insights powered by OpenAI.
- **Race Countdown**: Shows the time remaining until the next race.
- **Race Schedule**: Lists upcoming races with dates and times.
- **Driver and Constructor Standings**: Displays current standings for both drivers and constructors.
- **Last Race Results**: Shows results from the most recent race.
- **Admin Panel**: Allows backend management and data synchronization.

# File Description Inventory
- **docs/**: Contains documentation files, including deployment guides.
- **src/components/**: Reusable UI components for dashboards, user interface elements, and loading states.
- **src/contexts/**: Contains `AuthContext.tsx` for managing authentication state.
- **src/pages/**: Application pages including Index, AdminPage, LoginPage, SignupPage, ProfilePage, and SettingsPage.
- **src/App.tsx**: Main component for setting up application routing and context providers.
- **src/lib/**: Contains services for data fetching and Supabase operations.

# Technology Stack
- **React**: Frontend framework for building user interfaces.
- **TypeScript**: Provides type safety for the codebase.
- **Vite**: Build tool for fast development.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **React Router**: For managing routing and navigation.
- **Tanstack Query**: For data fetching and state management.
- **Supabase**: For authentication and database management.

# Usage
1. **Install Dependencies**: Run `pnpm install` to install all necessary packages.
2. **Initialize Backend Data**: Navigate to the `/admin` page after starting the development server and click "Sync F1 Data" to populate the database, then click "Generate Predictions."
3. **Start Development Server**: Use `pnpm run dev` to start the application.
4. **Build for Production**: Execute `pnpm run build` to create a production build.
