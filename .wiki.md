# Project Summary
The project is an F1 analytics application that provides comprehensive insights into Formula 1 racing data. It includes user account management features such as login, signup, profile, and settings, along with a dashboard that displays race statistics and standings. The application aims to enhance the user experience by offering detailed analytics and easy navigation.

# Project Module Description
- **User Account Management**: Features login, signup, profile management, settings, and password recovery.
- **Dashboard**: Visualizes race statistics, standings, and other relevant data.
- **Routing**: Handles navigation across different views of the application.

# File Description Inventory
- **docs/**: Contains documentation files, including design and architecture diagrams.
- **src/components/**: Includes reusable UI components such as buttons, cards, forms, and layout components.
- **src/pages/**: Contains main application pages like LoginPage, SignupPage, ProfilePage, etc.
- **src/App.tsx**: The main component responsible for setting up application routing and context providers.
- **src/components/layout/Header.tsx**: The header component that provides navigation and search functionality.

# Technology Stack
- **React**: Frontend framework for building user interfaces.
- **TypeScript**: Superset of JavaScript that provides type safety.
- **Vite**: Build tool for faster development.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **React Router**: For managing routing and navigation.
- **Tanstack Query**: For data fetching and state management.

# Usage
1. **Install Dependencies**: Run `pnpm install` to install all necessary packages.
2. **Build the Application**: Execute `pnpm run build` to create a production build.
3. **Run the Application**: Use `pnpm run start` to start the development server.
