# Project Summary
The project is an F1 analytics application designed to provide users with insights into Formula 1 racing data. It features user account management, including login, signup, profile, and settings pages, along with a dashboard for displaying race statistics and standings.

# Project Module Description
- **User Account Management**: Includes pages for login, signup, profile, settings, and forgot password functionalities.
- **Dashboard**: Displays race statistics, standings, and other relevant data.
- **Routing**: Manages navigation between different application views.

# File Description Inventory
- **docs/**: Contains various documentation files including design and architecture diagrams.
- **src/components/**: Contains reusable UI components like buttons, cards, and forms.
- **src/pages/**: Contains the main application pages such as LoginPage, SignupPage, ProfilePage, etc.
- **src/App.tsx**: The main component that sets up the application routing and context providers.

# Technology Stack
- **React**: Frontend framework for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for faster development.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **React Router**: For routing and navigation.
- **Tanstack Query**: For data fetching and state management.

# Usage
1. **Install Dependencies**: Run `pnpm install` to install all necessary packages.
2. **Build the Application**: Execute `pnpm run build` to create a production build.
3. **Run the Application**: Use `pnpm run start` to start the development server.
